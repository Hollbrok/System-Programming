# Binary tree of search

# How to build

```shell
    git clone https://github.com/Hollbrok/System-Programming.git
    cd System-Programming/4\ sem/Binary\ tree
```

## For compile

```shell
    make
```

## For valgrind

```shell
    make valgrind
```

## For run tests

```shell
    make run_tests
```

The output of tests will be in tests/report in the file __index.html__
